# gamefindme
## Description
A full-stack application to find gamers.

## Technologies Used
- Node.js
- Express.js
- React
- PostgreSQL
- Sequelize
- JWT Authentication

## Installation
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure environment variables in `.env`.
4. Run the server: `npm start`.
